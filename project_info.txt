# Project Info - BewegendeAnimaties

## Projectomschrijving
BewegendeAnimaties is een Python-tool voor het genereren van fMRI-stijl GIF animaties met wetenschappelijke authenticiteit en authentieke heatmap-visualisaties. Het project maakt gebruik van een statische hersenachtergrond waarop verschillende soorten bewegende elementen worden geanimeerd in karakteristieke fMRI-kleuren (oranje/rood/geel gloed) met uitgebreide neuroimaging kleurschalen, geavanceerde fMRI-realisme effecten en **NIEUW**: authentieke heatmap enhancement voor onmiddellijke herkenning als professionele fMRI-visualisaties.

## Project doel, nut en functie
**Doel**: Het creëren van visueel aantrekkelijke en wetenschappelijk authentieke animaties die hersenactiviteit simuleren met realistische fMRI-kleurschalen, geavanceerde fMRI-realisme effecten en **NIEUW**: authentieke heatmap enhancement die niet te onderscheiden zijn van echte fMRI-scans en onmiddellijk herkenbaar zijn als professionele fMRI-heatmaps.

**Doelgroep**: 
- Neurowetenschappers en onderzoekers (verhoogde herkenning van authentieke fMRI-visualisaties)
- Medische professionals en clinici (vertrouwde heatmap-stijl voor klinische geloofwaardigheid)
- Educatieve instellingen voor het uitleggen van hersenprocessen (studenten herkennen echte fMRI-scans)
- Wetenschappelijke presentatie- en publicatiedoeleinden (realistische fMRI-representatie)
- fMRI-software ontwikkelaars en visualisatie specialisten (referentie implementatie)

**Nut**: 
- Maakt complexe hersenprocessen visueel begrijpelijk met wetenschappelijke authenticiteit
- Ondersteunt wetenschappelijke communicatie met professionele geloofwaardigheid
- Biedt flexibele configuratie voor verschillende use cases
- Verhoogt herkenbaarheid en wetenschappelijke waarde door realistische visualisaties
- **ENHANCED**: Authentieke fMRI-kleurschalen verhogen wetenschappelijke geloofwaardigheid
- **ENHANCED**: fMRI-realisme effecten maken animaties ononderscheidbaar van echte fMRI-scans
- **ENHANCED**: Voxel-texturen, ruis patronen en spatial smoothing voor authentieke uitstraling
- **ENHANCED**: Statistical thresholding en cluster detectie zoals in echte fMRI-analyse
- **ENHANCED**: Anatomische variatie en temporele correlatie voor realistische hersenactiviteit
- **NIEUW**: Heatmap enhancement voor onmiddellijke herkenning als fMRI-heatmaps
- **NIEUW**: Authentieke heatmap blending modes (screen, overlay, multiply) zoals in fMRI-software
- **NIEUW**: Wetenschappelijk accurate kleurmapping conform neuroimaging standaarden
- **NIEUW**: Intensiteit-gebaseerde transparantie en vloeiende kleurovergangen
- **NIEUW**: Activatie hotspots en ruimtelijke gradiënten voor natuurlijke heatmap-look
- **NIEUW**: Diepte perceptie en 3D-achtige heatmap effecten

## Architectuur/modulestructuur
```
BewegendeAnimaties/
├── main.py                    # Centraal script voor alle animaties
├── demo_bewegend_mannetje.py  # Demo script voor bewegend mannetje
├── demo_heatmap_enhancement.py # Demo script voor heatmap enhancement ✅ NIEUW
├── test_enhanced_colors.py    # Test script voor enhanced fMRI kleuren ✅
├── requirements.txt           # Python dependencies
├── config/
│   └── constants.py          # Configuratie en instellingen ✅ (uitgebreid met heatmap enhancement)
├── animaties/
│   ├── __init__.py           # Package initialisatie ✅
│   ├── bewegend_mannetje.py  # Bewegend figuur animatie ✅ (enhanced + heatmap enhancement)
│   ├── vlekken_verschijnend.py # Uitspreidende vlekken ✅ (enhanced + fMRI-realisme)
│   ├── vlekken_verdwijnend.py  # Verdwijnende vlekken ✅ (enhanced + fMRI-realisme)
│   └── tekst_verschijnend.py   # Letter-voor-letter tekst ✅ (enhanced + fMRI-realisme)
├── utils/
│   ├── image_utils.py        # Afbeelding manipulatie ✅ (uitgebreid met heatmap compositing)
│   ├── color_utils.py        # fMRI kleurenschema ✅ (volledig vernieuwd + heatmap enhancement)
│   ├── heatmap_utils.py      # Heatmap enhancement utilities ✅ NIEUW
│   └── gif_utils.py          # GIF generatie utilities ✅
└── output/                   # Gegenereerde GIF bestanden
```
